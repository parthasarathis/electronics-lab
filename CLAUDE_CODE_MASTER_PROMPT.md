## ADAPTIVE LAYOUT SYSTEM — RESPONSIVE BREAKPOINTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Every device HTML must include this responsive layout system:

MOBILE  (< 640px)  → Single column. 3D canvas top 36vh. Panels scroll below.
TABLET  (640-1024px) → Two-column. 3D canvas left 50%. Panels scroll right.
DESKTOP (> 1024px) → Three-column. 3D canvas left 40%. Panels center. IV curve right.

CSS to include in EVERY device file:
```css
/* ── RESPONSIVE LAYOUT ENGINE ── */
:root {
  --col-layout: 1;   /* overridden by media queries */
  --canvas-h: 36vh;
  --accent: #00aaff; /* overridden per category */
}

/* MOBILE default */
#app {
  display: flex;
  flex-direction: column;
  height: 100vh;
  overflow: hidden;
}
#canvas-section {
  flex-shrink: 0;
  height: var(--canvas-h);
  position: relative;
}
#panels-section {
  flex: 1;
  overflow-y: auto;
  padding: 8px 10px;
  display: flex;
  flex-direction: column;
  gap: 8px;
}

/* TABLET (640px+) */
@media (min-width: 640px) {
  #app {
    flex-direction: row;
    flex-wrap: wrap;
  }
  #header {
    width: 100%;
    flex-shrink: 0;
  }
  #canvas-section {
    width: 50%;
    height: calc(100vh - 52px);
  }
  #panels-section {
    width: 50%;
    height: calc(100vh - 52px);
    overflow-y: auto;
  }
  /* Slightly bigger text */

  .ptitle { font-size: 9px !important; }
}

/* DESKTOP (1024px+) */
@media (min-width: 1024px) {
  #canvas-section {
    width: 42%;
  }
  #panels-section {
    width: 35%;
  }
  #curve-section {
    width: 23%;
    height: calc(100vh - 52px);
    overflow-y: auto;

  }

  /* On desktop they move to right column via JS or duplicated */
```
